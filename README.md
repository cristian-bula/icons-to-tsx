<h1 align="center" id="title">Icons to tsx</h1>

<p id="description">Script to optimize svg icons and convert them to tsx or jsx</p>

  
  
<h2>🛠️ Installation Steps:</h2>

<p>1. Install packages</p>

```
npm i
```

<p>2. Run project</p>

```
npm run build
```
